# Primeros Pasos

<Alert type="info">
  Esta es la primera versión de MdEditor. Posiblemente encuentres errores. Ten encuenta que esto es un proyecto personal y no está destinado a ser un producto completo por el momento. Eso si, Se mantendra actualizado. 
</Alert>

Esta versión inicial incluye las funcionalidades básicas esenciales de un editor de documentos para formatos como Markdown.


## Características principales

- **Editor de texto**: Un editor limpio y minimalista optimizado para escribir en Markdown.
- **Vista previa en tiempo real**: Visualiza instantáneamente cómo se verá tu documento formateado.
- **Soporte para sintaxis básica**: 
  > - Encabezados
  > - Listas ordenadas y no ordenadas
  > - Enlaces
  > - Imágenes
  > - Texto en negrita e itálica
  > - Bloques de código y en Linea
  > - Tablas
  > - Separadores
  > - Citas
- **Shortcuts para:**
  > - itálica (ctrl + i)
  > - Negrita (ctrl + b)
  > - Enlaces (ctrl + k)
- **Autoguardado del contenido de la sesión.**
- **Posibilidad de Descargar el archivo**
- **Temas de la vista previa:**
  > - Github
  > - Dracula 
  > - Monokai
  > - Nord

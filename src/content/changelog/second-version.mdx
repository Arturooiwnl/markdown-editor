# Novedades y Mejoras

<Alert type="success">
  ¡MdEditor evoluciona! Esta actualización trae consigo nuevas características centradas en la gestión y persistencia de documentos.

Esta versión introduce funcionalidades importantes para mejorar la experiencia de usuario y la gestión de documentos.

## Características principales

- **Autenticación**: 
  > - Se migro la autenticación de Clerk a GitHub con Firebase Auth.

- **Gestión de Documentos**:
  > - Guardado de documentos en la nube.
  > - Límite de 5 documentos por usuario (Por el momento).
  > - Edición y actualización de documentos.
  > - Eliminación de documentos.
  > - Títulos y descripciones personalizadas.

- **Mejoras en la Interfaz**:
  > - Se modifico el selecor de temas para que sea global (Editor Y vista previa). 
  > - Nueva sección de documentos guardados.
  > - Modal de guardado mejorada.
  > - Toasts para notificaciones.
  > - Se adapto la seccion de Changelog para no refrescar la página.
  > - Se resolvieron errores con colores.
